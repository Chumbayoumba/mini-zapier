2. Inject: `PrismaService`, `CronService`
3. Three `@OnEvent` handlers:

**`@OnEvent('workflow.activated') handleActivated({ workflowId })`:**
- Load workflow by id from Prisma
- Parse `definition.nodes` to find the trigger node: `n.type === 'triggerNode'` OR `['WEBHOOK', 'CRON', 'EMAIL'].includes(n.data?.type)`
- If no trigger node found, return early
- Extract `triggerType = triggerNode.data.type` and `config = triggerNode.data.config || {}`
- Upsert Trigger record: `prisma.trigger.upsert({ where: { workflowId }, create: { workflowId, type: triggerType, config, isActive: true, webhookToken: triggerType === 'WEBHOOK' ? randomUUID() : null }, update: { type: triggerType, config, isActive: true } })`
- IMPORTANT for upsert update: Do NOT overwrite webhookToken if it already exists — only set on create
- If `triggerType === 'CRON'`, call `this.cronService.scheduleCron(trigger.id, config, workflowId)`
- Log: `Trigger synced for workflow ${workflowId}: ${triggerType}`

**`@OnEvent('workflow.deactivated') handleDeactivated({ workflowId })`:**
- Find trigger by workflowId
- If not found, return early
- Update trigger: `isActive: false`
- If trigger.type === 'CRON', call `this.cronService.stopCron(trigger.id)`

**`@OnEvent('workflow.updated') handleUpdated({ workflowId })`:**
- Load workflow, find trigger node in definition (same logic as handleActivated)
- If no trigger node, return early
- Find existing trigger record. If none exists, return early (only sync if trigger was previously activated)
- If trigger exists and is active: update config, and if CRON, stop old + schedule new
- Use `prisma.trigger.update({ where: { workflowId }, data: { type: triggerType, config } })`
- If `trigger.type === 'CRON'` or `triggerType === 'CRON'`: `cronService.stopCron(trigger.id)` then if `isActive`: `cronService.scheduleCron(trigger.id, config, workflowId)`

Import `randomUUID` from `crypto`.

**2. Update `apps/backend/src/triggers/webhook/webhook.service.ts`:**

Replace `EngineService` injection with `QueueService` injection:

```typescript
import { Injectable, NotFoundException, Logger } from '@nestjs/common';
import { PrismaService } from '../../prisma/prisma.service';
import { QueueService } from '../../queue/queue.service';

@Injectable()
export class WebhookService {
  private readonly logger = new Logger(WebhookService.name);

  constructor(
    private prisma: PrismaService,
    private queueService: QueueService,
  ) {}

  async processWebhook(token: string, body: any, headers: Record<string, string>) {
    const trigger = await this.prisma.trigger.findUnique({
      where: { webhookToken: token },
      include: { workflow: true },
    });

    if (!trigger || trigger.workflow.status !== 'ACTIVE') {
      throw new NotFoundException('Invalid or inactive webhook');
    }

    this.logger.log(`Processing webhook for workflow ${trigger.workflowId}`);

    const jobId = await this.queueService.addExecution(trigger.workflowId, {
      body,
      headers,
      receivedAt: new Date().toISOString(),
    });

    await this.prisma.trigger.update({
      where: { id: trigger.id },
      data: { lastTriggeredAt: new Date() },
    });

    return { jobId, status: 'triggered' };
  }
}
```
