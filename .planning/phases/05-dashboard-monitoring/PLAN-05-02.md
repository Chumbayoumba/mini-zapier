## Implementation Steps

### Task 1: Add Stats API Hook
**File**: `apps/frontend/src/hooks/use-executions.ts`
- Add `useStats()` hook:
  ```typescript
  export function useStats() {
    return useQuery({
      queryKey: ['stats'],
      queryFn: () => api.get('/stats').then(r => r.data),
      refetchInterval: 30000, // fallback polling every 30s
    });
  }
  ```
- Add `useChartData(period: '7d' | '30d')` hook:
  - Fetch executions for the period
  - Group by day using `date-fns` `format(date, 'yyyy-MM-dd')`
  - Return `{ date: string, success: number, failed: number, total: number }[]`
  - Use `useMemo` to avoid recomputation on re-renders

### Task 4: Wire useDashboardLive Hook
**File**: `apps/frontend/src/hooks/use-websocket.ts`
- Add `useDashboardLive()` hook:
  ```typescript
  export function useDashboardLive() {
    const queryClient = useQueryClient();
    
    useWebSocket({
      events: {
        'execution:completed': () => {
          queryClient.invalidateQueries({ queryKey: ['stats'] });
          queryClient.invalidateQueries({ queryKey: ['executions'] });
          queryClient.invalidateQueries({ queryKey: ['chart-data'] });
        },
        'execution:started': () => {
          queryClient.invalidateQueries({ queryKey: ['stats'] });
        },
      },
    });
  }
  ```
- Leverage existing `useWebSocket` hook — just subscribe to relevant events
- On each event, invalidate React Query caches so dashboard auto-refreshes

### Task 5: Rewrite Dashboard Page
**File**: `apps/frontend/src/app/(dashboard)/dashboard/page.tsx`
- Remove all inline stats cards and hardcoded data
- Import and compose:
  ```tsx
  export default function DashboardPage() {
    const { data: stats, isLoading: statsLoading } = useStats();
    const [chartPeriod, setChartPeriod] = useState<'7d' | '30d'>('7d');
    const { data: chartData, isLoading: chartLoading } = useChartData(chartPeriod);
    
    useDashboardLive(); // WebSocket real-time updates
    
    return (
      <div className="space-y-6">
        <h1>Dashboard</h1>
        <StatsCards {...stats} isLoading={statsLoading} />
        
        <div className="grid grid-cols-1 lg:grid-cols-2 gap-6">
          <ExecutionBarChart data={chartData} isLoading={chartLoading} />
          <WorkflowStatusChart data={stats} isLoading={statsLoading} />
        </div>
        
        <div className="grid grid-cols-1 lg:grid-cols-2 gap-6">
          <ExecutionLineChart 
            data={chartData} 
            period={chartPeriod} 
            onPeriodChange={setChartPeriod}
            isLoading={chartLoading} 
          />
          <RecentExecutions />
        </div>
      </div>
    );
  }
  ```
- Page layout: stats row → charts 2-col grid → line chart + recent executions 2-col grid
- Use existing card/container styling from the design system
