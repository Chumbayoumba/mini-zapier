FlowForge (Mini-Zapier) is a monorepo workflow automation SaaS. It provides a visual drag-and-drop workflow editor, trigger/action execution engine, integrations, executions history, dashboard analytics, notifications, and auth. Tech stack: Turborepo + pnpm monorepo with apps/backend (NestJS 10, Prisma, PostgreSQL, BullMQ, Redis, Socket.io) and apps/frontend (Next.js 15 App Router, React 19, TanStack Query, Zustand, React Flow, Tailwind, shadcn/ui). Production app URL is documented in README as https://zapier.egor-dev.ru and API docs at /api/docs.
